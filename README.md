## 🚀 **Installation Guide**

1️⃣ Open **Extensions** in VS Code.
2️⃣ Search for **`Akashic Theme Suite`**.
3️⃣ Click **Install**.
4️⃣ Open Command Palette:

- **Windows/Linux:** `Ctrl + Shift + P`
- **Mac:** `⌘ + Shift + P`
  5️⃣ Select **`Preferences: Color Theme`**.
  6️⃣ Pick your variant:
- `Akashic`
- `Akashic Aurora`
- `Akashic Daybreak`
  7️⃣ 💖 **Enjoy your new coding vibe!**
